# GDSC_GM1.github.io
Proyecto sobre la base de datos GDSC realizado en la asignatura de CIPDB por Alma Gavilán Velasco, Cristina Gómez Baena y Reyes Mª Jurado Jiménez.

La base de datos GDSC cuenta con datos de 1000 líneas celulares humanas de cáncer tratadas con cientos de compuestos. Contiene datos sobre la respuesta a fármacos y marcadores genéticos de sensibilidad. El objetivo de esta base de datos es mejorar los tratamientos frente al cáncer, ayudando a identificar biomarcadores terapeúticos que se pueden asociar a patrones de respuesta ante fármacos anticancerígenos.

Para visualizar correctamente este proyecto en Rstudio es necesario tener instaladas y cargadas las librerías: readxl, dplyr, knitr, ggplot2 y tidyverse. El proyecto se realizó con la versión de R 4.4.3.
